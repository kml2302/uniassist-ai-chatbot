# UniAssist - AI University Chatbot 🎓🤖

UniAssist is an AI-powered chatbot designed to help students with university-related queries like admissions, courses, and campus information.

## 🚀 Features
- Smart responses to student queries
- Course registration guidance
- Clean and user-friendly UI
- Fast and automated replies

## 🛠 Tech Stack
- Python (Flask)
- HTML
